# Sample REST Service

A simple Clojure REST service sample.

## Prerequisites

You will need [Leiningen][] 2.0.0 or above installed.

[leiningen]: https://github.com/technomancy/leiningen

## Development

Run tests:

    lein run
    dev=> (refresh-and-test)


## Running

To start the service run:

    lein run
